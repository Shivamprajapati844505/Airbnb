# Airbnb Clone 🏡

## Airbnb is a full-fledged MVC-based hotel room booking website built using Node.js, Express, and EJS. It allows users to browse, book, and manage hotel rooms with an intuitive and interactive UI.

## 🚀 Features
- 🌍 **User Authentication** - Secure login and registration.
- 🏡 **Property Listings** - Users can add, edit, and delete listings.
- 🔍 **Search & Filters** - Users can search for listings based on location, price, and availability.
- 🛒 **Booking System** - Integrated booking and reservation system.
- 💬 **Reviews & Ratings** - Users can leave reviews and ratings.
- 📸 **Image Upload** - Cloudinary integration for uploading property images.

## 🛠️ Tech Stack
- **Frontend:** EJS (Embedded JavaScript)
- **Backend:** Node.js, Express.js
- **Database:** MongoDB, Mongoose
- **Authentication:** Passport.js, Bcrypt.js
- **File Uploads:** Cloudinary
- **Other Tools:** Middleware, dotenv, Express-session

## 📦 Installation & Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/Shivamprajapati844505/Airbnb-clone.git
   cd airbnb-clone
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Configure environment variables in a `.env` file.
4. Start the development server:
   ```bash
   npm start
   ```
   This will launch the application at `http://localhost:3000/`.
